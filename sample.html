Jump Game Solution

This page provides a solution to the Jump Game problem, where you need to determine if you can reach the last index of the array given the maximum jump lengths at each position.

Python Code


def canJump(nums):
    max_reachable = 0
    for i in range(len(nums)):
        if i > max_reachable:
            return False
        max_reachable = max(max_reachable, i + nums[i])
        if max_reachable >= len(nums) - 1:
            return True
    return False

Explanation

To solve this problem, we use a greedy approach:

  1. Initialize max_reachable to 0. This variable will keep track of the farthest index we can reach.
  2. Iterate through the array:
    • If the current index i is greater than max_reachable, return false because it means we cannot reach this index.
    • Update max_reachable to be the maximum of its current value and i + nums[i].
    • If max_reachable is greater than or equal to the last index, return true because we can reach or exceed the last index.
  3. If we complete the loop without returning true, return false.

Example

Consider the array [2, 3, 1, 1, 4]:

  • Start with max_reachable = 0.
  • At index 0, nums[0] = 2. Update max_reachable to 2.
  • At index 1, nums[1] = 3. Update max_reachable to 4.
  • At this point, max_reachable (4) is greater than or equal to the last index (4), so return true.
